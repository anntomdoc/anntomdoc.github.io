class Essentials.Data3D_BigData.Data3D_BigData(address, slices_to_consider_3d=5)¶

Bases: object

This class is designed to handle 3D dicom samples in python 2. Tested on python 2.7. Note: Making an instance of this class does not load samples. It just scrolls through the address folder and subfolders and prepares self.samples_addresses. self.samples_addresses is a dictionary which containes the addresses of the dicom files in each subfolder. All dicom addresses for sample sample_number are stored in self.samples_addresses[‘Sample’+str(sample_number)].

address: string.
Address of the folder including all the dicom samples. dicom files might be in subfolders. Each subfolder including dicom files will be considered as a single 3D sample.
slices_to_consider_3d: integer (Default value is 5).
Minimum number of slices in a sample to be considered as 3D. All subfolders with more than slices_to_consider_3d samples will be considered as a 3D sample.
